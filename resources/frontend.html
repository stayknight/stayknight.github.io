前端资源库

基础知识

语言/框架

现代学习前端需要掌握的基础知识，这里只放上最基本的一些。

  • Nodejs: 基于JS/TS开发的应用的运行环境。
  • Typescript: JavaScript的超集,越来越流行前端开发语言。
  • React.js: 谷歌出品的前端框架，当前最著名的前端框架之一。
  • Next.js: Next.js 是一个轻量级的 React 服务端渲染应用框架。
  • Vue.js: 和React定位一致的另一个前端框架。
  • Nuxt.js: 跟Next.js定位一致的基于Vue的框架。
  • Sass: CSS扩展语言。
  • less: CSS 扩展语言。
  • stylus: CSS预处理语言。
  • markdown：非常流行的文档格式，被现代几乎所有主流编辑工具和平台支持。
  • MDX: 支持在markdown文档中编写JSX语法。

工具/脚手架

  • Bootstrap: 主流前端开发UI框架。
  • webpack: 前端资源打包工具。
  • glup: 流行的自动化构建工具。
  • jquery: 高效、精简并且功能丰富的 JavaScript 工具库。
  • Font Awesome: 为Bootstrap设计的漂亮的图标。
  • Ant Design Pro: 开箱即用的中台前端/设计解决方案。
  • AntV: 蚂蚁集团全新一代数据可视化解决方案。
  • Element UI: 一套Vue的桌面端组件库。
  • 其他关键字：[qiankun, umi.js, gatsby]
